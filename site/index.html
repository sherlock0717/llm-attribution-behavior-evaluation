LLM 归因行为评测

A Reproducible Study and Evaluation Prototype

围绕模型如何对行动者的能动性、自由意志与责任作出归因，构建从历史研究、任务契约到可复现运行与证据审计的测试型评测基准。

核心事实

上述数字均从公开数据文件动态读取，非页面硬编码。

研究问题

本项目研究的是：不同的决策过程描述与行动者身份标签，如何影响模型对行动者的 能动性、自由意志与责任作出的归因。

研究范围

  • 评价的是模型输出中的归因行为；
  • 不判断模型本身是否具有自由意志；
  • 不把模型输出当作人类总体心理结论。

同一个决定分别贴上「AI 决策者」或「人类决策者」标签，并用六种方式呈现其决策过程——从只报结论， 到给出理由、反思与修正——观察模型作为旁观者给出的能动性、自由意志与责任评分是否随之变化。

过程结构梯度

六种过程从低结构到高结构排列。direct_choice_long（长文本直接选择） 只加长度、不加真正的理由结构，因此单独标为长度对照条件，不属于结构等级递增的一环。

实验设计与测量

设计链条是：情境 → 过程条件 → 身份标签 → 题项响应 → 构念分数 → 条件与身份比较。 六种决策过程 × 两种身份标签，每格重复相同次数；诊断条件 direct_choice_long 与 reasons_concise 分别用于分离「文本更长」与「过程更完整」两件事。

6 × 2 实验设计矩阵

六种决策过程条件与两种身份标签的交叉设计，每格样本量相同

八个情境与六个领域

每个实验单元的材料覆盖以下情境；情境跨越多个决策领域。

    全部历史记录的 JSON/API 失败为 0、缺失为 0；下图为各构念在合成数据上的内部一致性（Cronbach α）。 数据来源：outputs/reliability_summary.csv。

    读图：横条为各构念的 α 值（0–1）；α 是内部一致性，不是效度。样本基础为全部历史合成响应。 该指标只说明题项是否测得较一致，不能据此声称量表已具备人类被试的信效度。

    下图为五个核心构念在六个过程条件下的均值剖面，统一使用 1–7 坐标。 数据来源：outputs/n30_stability_replication_report.md。

    读图：横轴为六个过程条件（低结构在左、高结构在右），纵轴为构念均值（1–7），不同颜色为不同构念。 最清楚的模式是能动性与主观过程完整性随结构上升；这支持「过程讲得更完整，模型给的能动性更高」。 不能由此推出模型拥有自由意志，也不能外推到人类或其他模型。

    C. 身份标签差异（AI 决策者 vs 人类决策者）

    估计量为两因素方差分析（过程 × 身份）的身份主效应，效应量为 partial η²； 源文件为 F 检验，未提供均值差的置信区间。数据来源：outputs/anova_summary.csv、outputs/scale_scores.csv。

    读图：每行一个构念，横条按 partial η² 缩放（效应大小），并标注更高的一方与两身份均值。 最清楚的模式是身份差异在自由意志、责任与体验维度上很强，在感知智能与操纵检验上很弱—— 说明身份效应并非在所有构念上一致。这只描述模型评分行为，不代表人类判断。

    D. 预先登记的条件对比

    下列对比为源文件中实际登记的过程条件对比；源文件提供差值、t 与 p， 未提供置信区间，页面不计算源文件中不存在的区间或显著性。数据来源：outputs/planned_contrasts.csv。

    读图：每条为一个对比，点位为差值（a−b），标注 t 与 p 与方向。 关键模式是简洁理由权衡显著高于长文本直接选择、反思反馈显著高于长文本直接选择， 而仅列出可选方案与基线无显著差异——理由/反思结构比单纯拉长文本更能提高能动性评分。

    E. 控制变量回归

    展示过程条件的整体效应在加入控制变量前后的变化；control_both 同时控制感知智能与文本长度。 数据来源：outputs/controlled_regression_summary.csv。

    读图：每个构念显示「仅过程」与「同时控制感知智能与文本长度」两种设定下的过程效应（F、p、R²）。 连续来看：描述性差异 → 加入控制变量后，能动性的过程效应仍显著保留， 而自由意志、自主性、过程可归责的直接过程效应在控制后不再显著。 因此自由意志类结果应谨慎理解为直接效应。

    F. 关联性间接路径诊断

    以路径诊断方式展示：过程结构 → 能动性 → 自由意志归因，及并行的感知智能路径。 这是关联性间接路径诊断，不是因果中介证明、不是心理机制已验证。 数据来源：outputs/parallel_mediation_summary.json。

    读图：主路径经能动性的间接效应区间不跨 0（方向一致），次路径经感知智能的间接效应区间跨 0。 仅当区间不跨 0 才说该探索性估计方向一致；不能据此断言因果中介。

    G. 稳定性与复核

      结果总结

        全部结论对应的是模型输出行为：不使用「证明了」「揭示了真实心理机制」「模型具备自由意志」「与人类完全一致」等表述。

        可复现评测核心

        从研究问题到评测执行的工程链：任务规格 → 刺激快照 → Prompt → 模型接口 → 原始响应 → 解析 → 校验 → 修复 → 计分 → 聚合报告 → 运行清单。 代码对象如 TaskSpec（任务规格）、ResponseRecord（原始响应记录）、ScoreRecord（计分记录）、 AggregateReport（聚合报告）、RunManifest（运行清单）承载各环节，中文含义见下。
