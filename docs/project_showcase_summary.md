# 项目展示：AI 决策过程如何影响自由意志归因？

## 这个项目解决什么问题

当一个 AI Agent 给出决定时，人们会不会因为它“解释了理由”“比较了选项”“进行了反思”而更倾向于认为它有 agency，甚至更像是在自由地行动？

这个项目用 LLM-simulated respondents 做了一个模拟预实验，用来检查实验材料、理论模型和分析流程是否可行。

## 为什么它是心理学 + 哲学 + AI 交叉

- 心理学：研究人们如何感知 agency、心智和责任。
- 哲学：涉及自由意志、理由响应性和责任归因。
- AI：对象是 AI Agent 的决策过程呈现方式。

项目不讨论 AI 是否真的有自由意志，而是研究观察者如何根据决策过程进行归因。

## 我做了什么

我构建了一个 6 × 2 的模拟实验：

- 6 种决策过程：直接选择、长文本直接选择、候选方案、简洁理由、完整理由、反思反馈。
- 2 种身份标签：AI 决策者、人类决策者。

我还加入了两个诊断条件：

- `direct_choice_long`：检查是不是文本越长评分越高。
- `reasons_concise`：检查短文本中的理由结构是否仍有效。

## 技术流程

1. 用 Python 生成实验材料。
2. 调用 DeepSeek API 模拟问卷参与者。
3. 输出 JSONL 和 CSV。
4. 计算量表均分、信度、ANOVA、控制回归和中介模型。
5. 加入并行竞争中介：agency vs perceived intelligence。
6. 生成 Markdown 报告和图表。

## 主要发现

在 n-per-cell=30 的 LLM 模拟预实验中：

- factual check 稳定，说明 6 类材料可被模型区分。
- agency 是最稳定的主结果。
- 控制文本长度和感知智能后，决策过程仍显著影响 agency。
- 自由意志归因的直接效应不稳定，但 agency 的间接中介路径稳定。
- perceived intelligence 没有解释主要间接效应。
- 单纯列出候选方案不足以明显提高 agency 或自由意志归因。
- 理由权衡和反思反馈更像关键线索。
- 责任归因不稳定，只适合作为探索性结果。

## 研究边界

这是 LLM-simulated respondents 的模拟预实验，不是真实人类被试研究。它不能证明 AI 有自由意志，不能替代正式心理学信效度检验，也不能声称获得了正式心理学结论。

更准确的说法是：这个项目完成了材料预演、模型诊断和分析流程验证。

## 后续计划

下一步可以基于当前冻结版本开展真实人类被试预测试：

- 做专家内容效度评估。
- 检查题项理解度。
- 重新检验信度和因素结构。
- 验证 agency、perceived intelligence 和自由意志归因是否能区分。
- 在真实样本中复核中介模型。
