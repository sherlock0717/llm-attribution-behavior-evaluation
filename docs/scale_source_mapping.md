# 题项与理论来源映射

本文件逐项记录 `src/scales.py` 中的题项来源。当前题项池基于既有理论与量表构念进行情境化改写，并非对原量表的完整直接使用；除非未来完整直接采用原量表并遵守其计分方式，否则所有题项都视为“基于既有理论与量表构念的情境化归因题项池”，原研究的信度、效度和因素结构不能直接继承。

完整参考文献见文末「完整参考文献」一节，以及 `docs/research_and_measurement_sources.md`。

后续验证要求适用于全部题项：专家内容效度、真实被试预测试、Cronbach alpha / McDonald omega、EFA / CFA、区分效度、AI / 人类标签下的测量等价性。

## 来源分类（source_role）与使用方式

| source_role | 含义 | 涉及构念 | 是否为直接原题 | 是否可继承原信效度 |
|---|---|---|---|---|
| construct_adapted | 参考量表或理论中的构念，题项经过情境化改写 | agency、experience、free_will_attribution、autonomy、outcome_accountability、moral_praise_blame、process_accountability、perceived_intelligence | 否 | 否 |
| newly_written_manipulation_check | 本研究自编操纵检验，无外部量表来源 | factual_manipulation_check、subjective_process_completeness | 不适用 | 不适用 |

autonomy 目前只对应自主性与行动控制的一般理论背景（general_theoretical_background），在找到明确、直接采用的量表或理论前不指定具体量表。
