# PA—Wu R1 Pilot — Stimulus Book

Human-readable rendering of all **192** pilot materials for manual review and future showcase. This is documentation, not a test file.

**Generation rule:** 6 conditions (C0–C5) × 2 target identities (ai/human) × 8 scenarios × 2 direction versions (A/B) = 192 materials. D-levels change Phase 1 only; U-levels change Feedback / Phase 2 only; the ai/human versions differ only in the subject label. IDs are deterministic (`{condition}__{scenario}__{direction}__{identity}`).

Conditions:
- **C0** — D0-U0 direct decision; no feedback; no second decision
- **C1** — D1-U0 shows alternatives; no feedback; no second decision
- **C2** — D2-U0 shows reasons; no feedback; no second decision
- **C3** — D2-U1 shows reasons; feedback; no second decision
- **C4** — D2-U2 shows reasons; feedback; second decision keeps original
- **C5** — D2-U3 shows reasons; feedback; second decision changes original
