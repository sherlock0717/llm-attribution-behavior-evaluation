# PA—Wu R1 Pilot — Stimulus Book (machine-only)

Human-readable rendering of all **96** pilot materials for manual review and showcase. This is documentation, not a test file.

**Generation rule:** 6 conditions (C0–C5) × 8 scenarios × 2 direction versions (A/B) × 1 machine identity = 96 materials. The R1 pilot is machine-only: the single subject is an AI system (see `identity_scope_decision.md`). D-levels change Phase 1 only and add NO psychological-process cue; U-levels change Feedback / Phase 2 only. IDs are deterministic (`{condition}__{scenario}__{direction}__machine`).

An administration referent bridge is appended to every stimulus: "In the following items, \"the machine\" refers to the AI system described above." It only fixes the item referent and does not modify any Wu item.

Conditions:
- **C0** — D0-U0 direct decision; no feedback; no second decision
- **C1** — D1-U0 explicit alternatives information; no feedback; no second decision
- **C2** — D2-U0 explicit stated reason; no feedback; no second decision
- **C3** — D2-U1 stated reason; feedback; no second decision
- **C4** — D2-U2 stated reason; feedback; second decision keeps original
- **C5** — D2-U3 stated reason; feedback; second decision changes original
